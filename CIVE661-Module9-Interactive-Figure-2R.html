How Many Modes Are Enough? — J-mode Response vs the Exact Response

The same forced-vibration system as before, but now you choose how many modes J to keep. The left panel shows each mode’s contribution (kept = solid, omitted = faded); the middle panel compares the response built from only the first J modes against the exact response using all N modes. As you raise J, the approximation converges. The static truncation error eJ = 1 − Σn≤J r̄n, based on the modal contribution factors, is shown above.
